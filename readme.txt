The installation folder contains the following archives:

1. `com_comparisonchart_2.2.0.zip` - the installation archive, 
   that installs the component

2. `plg_content_comparisonchart.zip` - the installation archive for 
   CoparisonChart content plugin.

3. `plg_editors_xtd_comparisonchart.zip` - the installation archive for 
   CoparisonChart button plugin. This plugin adds special button into default HTML-editor
   for articles, so you can insert charts into articles.

4. mod_latest_charts.zip - the installation archive for 
   CoparisonChart Module.
